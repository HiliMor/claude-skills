---

The user wants to learn about a specific concept. Do two things:

1. **Explain it** clearly for a beginner
2. **Save it** to ~/claude-learning-log/

## How to Explain

- Use simple language, no jargon
- Use a real-world analogy if it helps
- Keep it short — under 10 lines
- Include one practical example

## How to Save

1. Create ~/claude-learning-log/ if it doesn't exist
2. Name the file: YYYY-MM-DD-learn-topic-in-kebab-case.txt
3. Write the entry in this format:

```
[YYYY-MM-DD] — topic
• Key point one. Example: short example.
• Key point two.
• Key point three.
```

4. Confirm to the user the filename it was saved to

$ARGUMENTS
