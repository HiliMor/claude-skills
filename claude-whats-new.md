Fetch the Claude Code changelog and produce a clear, plain-language summary of what's new.

## Instructions

1. Use WebFetch to load: https://code.claude.com/docs/en/changelog
2. Read the full page content
3. Summarize the findings — focus on:
   - New Claude models (name, capabilities, what improved)
   - New Claude Code features or CLI updates
   - API changes or new tools
   - Any deprecations or removals
4. List releases from newest to oldest

## Output Format

### What's New in Claude

#### Models
- List new models with key highlights

#### Claude Code / CLI
- List new features, commands, or behavior changes

#### API & Tools
- Notable API additions or changes
